# Neovim configuration

## Try out this config

Make sure to remove or move your current `nvim` directory

```
git clone https://github.com/Jayden1905/lua-nvim.git ~/.config/nvim
```

Run `nvim` and wait for the plugins to be installed.
You will notice treesitter pulling in and lsp language servers installing the next time you open Neovim.

**NOTE:** (This configuration isn't created by me. This is just a combination from different sources. I took mainly from [Christian Chiarulli](https://github.com/LunarVim/Neovim-from-scratch) & [Takuya Matsuyama](https://github.com/craftzdog/dotfiles-public).)

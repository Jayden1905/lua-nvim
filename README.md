This Neovim configuration is written in lua language. It is a combination of different configs from different sources. I took mainly from Christian Chiarulli and
Takuya Matsuyama.
Both of them are popular vim users and you can check their configs via youtube or their github repositories.
I mainly use this configuration for javascript and python development.

// Necessary installation
1.You need to install nodejs, npm, python3, pip3 or pip and a package manager depending on your operating system(I use brew since my os is macos).
2.You need to install code formatter for your language via npm or yarn.(e.g. prettier for javascript and black for python)(npm i -g prettier)(pip3 install black)
3.You need to install live-grep and lazyigit with your package manager (e.g. brew install lazygit, brew install ripgrep)

// Conclusion
This is entirely personal configuration, you can make changes depending on your preference. Thank you for checking out.
